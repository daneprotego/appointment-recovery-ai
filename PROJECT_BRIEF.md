# Appointment Recovery AI

Build a low-cost SaaS that reduces missed appointments and no-shows.

## Product
AI Appointment Recovery Platform

## Main Promise
Recover lost appointment revenue automatically.

## MVP
- User signup/login
- Business profile
- Appointment list
- Customer list
- SMS reminders
- AI SMS reply classifier
- Reschedule flow
- Waitlist fill logic
- Dashboard metrics
- Stripe pricing

## Pricing
- Free: $0
- Starter: $19/mo
- Growth: $39/mo
- Pro: $79/mo

## Stack
- Next.js
- TypeScript
- Tailwind
- Supabase
- Twilio
- OpenAI API
- Stripe
- Vercel
